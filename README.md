# Shiny ✨

A modern **freelance-matching web app** — built as a showcase by **[Governor Studio](https://governor-studio.vercel.app)**.

Answer a short survey about your project and get matched with the right tech freelances (web, back-end, design, mobile, data).

## Features
- 🧭 Multi-page app with client-side routing (React Router)
- 📝 Animated multi-step survey with progress and back/forward
- 🎯 Results page that recommends freelances from your answers
- 👥 Filterable freelance directory + detail profiles
- 🌗 Light / dark theme (persisted) · animated transitions (Framer Motion)
- 📱 Fully responsive, accessible

## Stack
**Vite · React 19 · React Router · Tailwind CSS v4 · Framer Motion**

## Run locally
```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
```
